# Jekyll - GitHub Pages - Einfache Blogs
## Vor- und Nachteile ### Nachteile - Generierte Seiten sind *statisch*, können nicht unmittelbar den Inhalt von Datenbanken widerspiegeln - Kein vom Client bedienbares CMS ### Vorteile - Einfache Bedienbarkeit durch Erstellung von Markdown Seiten - Flexible Erweiterbarkeit durch alle Möglichkeiten, die HTML und JavaScript Bieten - Auch Erweiterbar durch Jekyll-Plugins - Ermöglicht hohe Verfügbarkeit der Statisch generierten Seiten, Wenig Serverlast - Sind die Seiten einmal fertig ändern sie sich nicht - Kollaboration in Kombination mit Git Repositories
